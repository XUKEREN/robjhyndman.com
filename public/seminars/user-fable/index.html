Tidy forecasting in R

13 July 2018

Presentation at the useR 2018 conference in Brisbane, Australia

The forecast package in R is widely used and provides good tools for monthly, quarterly and annual time series. But it is not so well-developed for daily and sub-daily data, and it does not interact easily with modern tidy packages such as dplyr, purrr and tidyr.

I will describe our plans and progress in developing a collection of packages to provide tidy tools for time series and forecasting, which will interact seamlessly with tidyverse packages, and provide functions to handle time series at any frequency.

This talk is part of a series of three:

  1. Earo Wang. tsibble: The 15th time series standard
  2. Rob J Hyndman. Tidy forecasting in R
  3. Mitchell O’Hara-Wild. fasster: Forecasting multiple seasonality with state switching

To download the packages: tidyverts.org
